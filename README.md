# Logan Law Firm — Website (Local Preview)

This folder contains a Next.js (App Router) + Tailwind CSS project, with your photos optimized **locally** (no uploads).

## How to Run (Mac or Windows)
1) Install Node.js 18+ from https://nodejs.org/
2) Open your terminal:
   - **Mac:** Applications → Utilities → Terminal
   - **Windows:** Start → type `PowerShell` or `Command Prompt` → open
3) In the terminal, go to this folder. Example:
   - `cd Downloads/logan_site`  (replace with your actual path)
4) Install dependencies:
   ```
   npm install
   ```
5) Start the dev server:
   ```
   npm run dev
   ```
6) Open http://localhost:3000 in your browser.
